# 兢兢业业斗地主
## 项目简介
#### 本项目实现了Android端的四人斗地主小游戏，该系统面向18岁以上的用户，适用于18岁以上的各个年龄段。该系统的目的是让人们在空闲的时间以游戏的方式放松心情，该游戏不同于三人斗地主，增加了一些规则令人有种新鲜感，同时可以开启新的思路且充分开发大脑的灵活性。
